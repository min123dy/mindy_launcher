# mindy Launcher

**mindy Launcher**는 Microsoft 계정 로그인과 Minecraft 버전 관리 기능을 제공하는 Windows용 Minecraft 런처입니다.

## 주요 기능

* Microsoft 계정 로그인
* Minecraft 버전 설치 및 실행
* 게임 구매 여부 확인 및 Demo 모드 지원
* Java 경로 설정
* 메모리(RAM) 설정
* 창 크기 및 전체 화면 설정
* 게임 옵션 관리
* 스킨 다운로드 및 변경
* 런처 자동 업데이트
* 버전별 패치노트 확인

## 개발 환경

* C#
* .NET
* Windows Forms (Visual Studio)
* CmlLib.Core
* WebView2

## 빌드 방법

1. Visual Studio에서 솔루션(`.sln`) 파일을 엽니다.
2. 필요한 NuGet 패키지를 복원합니다.
3. **Release** 또는 **Debug** 구성으로 빌드합니다.
4. 생성된 실행 파일을 실행합니다.

## 다운로드

빌드된 최신 실행 파일은 **GitHub Releases**에서 다운로드할 수 있습니다.

프로젝트 페이지 우측의 **Releases** 또는 상단의 **Releases** 메뉴에서 최신 버전을 다운로드하세요.

## 업데이트

런처에는 자동 업데이트 기능이 포함되어 있습니다.

새 버전이 배포되면 런처에서 업데이트를 확인하고 최신 버전으로 설치할 수 있습니다.
